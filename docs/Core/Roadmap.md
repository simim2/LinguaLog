# Vision

LinguaLog는 AI 영어 일기 서비스가 아니라,

**사용자의 영어 성장 데이터를 기록하고 분석하는 플랫폼**을 목표로 개발합니다.

모든 Phase는 다음 목표를 향해 진행됩니다.

* 더 쉽게 기록한다.
* 더 많은 데이터를 축적한다.
* 더 좋은 인사이트를 제공한다.
* 언제 어디서나 사용할 수 있다.

# Long-term Vision

LinguaLog는 단순히 영어 일기를 저장하는 서비스가 아닙니다.

최종 목표는

**"나의 영어 성장 기록"**

을 만드는 것입니다.

사용자는

* 언제든 자신의 과거 영어를 돌아보고
* 현재 수준을 확인하고
* 성장 과정을 시각적으로 확인하며
* 앞으로의 목표를 세울 수 있어야 합니다.

---

# Development Strategy

모든 개발은 다음 순서를 따릅니다.

1. Writing Experience 개선
2. 데이터 구조 안정화
3. Dashboard 고도화
4. 성장 리포트
5. 데이터 활용 기능
6. 클라우드 전환
7. 모바일 앱 개발

---

# Guiding Principles

새로운 기능을 추가하기 전에 항상 아래 질문을 확인합니다.

* 사용자가 더 많이 기록할 수 있는가?
* 성장 데이터를 더 잘 축적할 수 있는가?
* Dashboard에서 활용 가능한가?
* 모바일에서도 재사용 가능한가?
* 기존 구조를 유지하는가?

위 조건을 만족하는 기능을 우선적으로 개발합니다.
